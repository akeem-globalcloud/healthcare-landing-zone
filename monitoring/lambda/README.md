# AWS Lambda

## Overview

AWS Lambda provides serverless event processing for the Healthcare Landing Zone.

Lambda receives events from Amazon EventBridge, processes the payload, enriches the event, logs execution details, and publishes notifications to Amazon SNS.

---

## Current Integration

- Amazon EventBridge
- Amazon SNS
- Amazon CloudWatch Logs

---

## Future Integrations

- GuardDuty
- AWS Config
- Security Hub
- AWS WAF
- Systems Manager

---

## Objectives

- Event-driven automation
- Serverless processing
- Security event enrichment
- Operational automation
- Automated remediation
