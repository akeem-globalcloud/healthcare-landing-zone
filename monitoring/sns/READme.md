# Amazon SNS Notification Service

## Overview

Amazon Simple Notification Service (SNS) provides centralized messaging and notification capabilities for the Healthcare Landing Zone.

SNS enables CloudWatch alarms and future AWS services to deliver real-time notifications to operations and security teams.

---

## Current Integration

- CloudWatch Alarms
- Email Notifications

---

## Future Integrations

- AWS Lambda
- Amazon EventBridge
- AWS GuardDuty
- AWS Security Hub
- AWS Config
- AWS WAF

---

## Objectives

- Real-time alerting
- Operational awareness
- Security notifications
- Event-driven automation
- Enterprise messaging
