# Amazon CloudWatch Monitoring

## Overview

Amazon CloudWatch provides centralized monitoring for the Healthcare Landing Zone.

CloudWatch collects metrics, logs, alarms, and dashboards that enable operations teams to monitor infrastructure health, detect issues, and support incident response.

---

## Components

- CloudWatch Dashboard
- CloudWatch Log Groups
- CloudWatch Metric Alarms

---

## Objectives

- Infrastructure Monitoring
- Security Monitoring
- Operational Visibility
- Compliance Monitoring
- Incident Detection
