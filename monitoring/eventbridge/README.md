# Amazon EventBridge

## Overview

Amazon EventBridge provides event-driven integration for the Healthcare Landing Zone.

It enables AWS services to publish events that are automatically routed to downstream targets such as Amazon SNS and AWS Lambda.

---

## Current Integration

- CloudWatch
- Amazon SNS

---

## Planned Integrations

- AWS Lambda
- GuardDuty
- Security Hub
- AWS Config
- CloudTrail
- AWS WAF

---

## Objectives

- Event-driven automation
- Centralized event routing
- Operational automation
- Security event processing
- Enterprise integration
